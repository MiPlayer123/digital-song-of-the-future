# Digital Song of the Future

*Perihelion Drift* — a fully synthesized song built from scratch in Python.
No DAW, no samples, no recorded instruments. Every kick, bass, pad, lead, and
vocal chop is generated from first principles — raw math turned into a 3:20
melodic-house track.

## Run it

```bash
pip install numpy scipy
python mars_perihelion_v2.py
```

Outputs `perihelion_drift_v2.wav` (~30 seconds to render on a modern laptop).

## Backstory

The song is set in **2073**, at the annual *Perihelion Festival* inside the
domes of Arcadia Planitia, Mars — a world where air is the most precious
shared resource and music plays only once a year, for everyone at once. The
production choices encode the setting: a 60 Hz life-support drone runs
underneath the whole track, and sidechain compression makes the song "breathe"
exactly 124 times per minute — once per kick, in sync with the audience.
